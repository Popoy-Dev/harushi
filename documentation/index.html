Other


Zoom image effect in Single Page.

Add a data-action="zoom" attribute to the images you want to make zoomable. For example:

<img src="img/blog_post_featured.png" data-action="zoom">

Making smart sticky elements.

Just call "stick_in_parent" on the elements you want to be stuck inside of their parent. Sticky elements “bottom out” so they never leave the container, no more worrying if a sticky element will accidentally cover your footer. For example:

$("#sidebar").stick_in_parent();
